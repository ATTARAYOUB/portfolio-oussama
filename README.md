## 🛠️ Development

### Local Setup

```bash
# Clone the repository
git clone https://github.com/ATTARAYOUB/oussama-kirene-portfolio.git

# Navigate to project
cd oussama-kirene-portfolio

# Open in browser
open index.html
```

### Customization

Edit `index.html` to:
- Update personal information
- Change colors (CSS variables in `:root`)
- Modify content and sections
- Add or remove skills/experience

### CSS Variables

Main theme colors can be customized in the `:root` section:

```css
:root {
  --accent: #d4a853;           /* Gold accent color */
  --bg-primary: #0a0a0b;       /* Dark background */
  --text-primary: #e8e8ec;     /* Light text */
  /* ... more variables */
}
```
